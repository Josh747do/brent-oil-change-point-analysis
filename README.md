# Brent Oil Price Change Point Analysis

Bayesian change point analysis of Brent oil prices to quantify the impact of geopolitical and economic events.

## Project Structure
- `data/raw/` - Original Brent oil price dataset
- `data/processed/` - Cleaned and transformed data
- `notebooks/` - Jupyter notebooks for analysis
- `src/` - Python source code modules
- `reports/figures/` - Generated charts and visualizations

## Setup
1. Clone repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
